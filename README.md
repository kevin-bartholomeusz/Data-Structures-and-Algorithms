# Java DSA Reference 📚

A collection of core data structures and algorithms implemented in Java — written and refined while studying Computer Science at university. This repo serves as both a personal reference and a reflection of concepts covered throughout the course.

---

## What's Inside

### 🔍 Searching
| File | Description |
|------|-------------|
| `LinearSearch_Array.java` | Linear search over a primitive array |
| `LinearSearch_LinkedList.java` | Linear search over a linked list |
| `BinarySearch.java` | Iterative binary search on a sorted array |
| `BST_Search.java` | Insert and search on a Binary Search Tree |

### 📊 Sorting
| File | Description |
|------|-------------|
| `SelectionSort.java` | Selection sort — finds max, swaps to end |
| `MergeSort.java` | Recursive merge sort with auxiliary array |

### 🔗 Linked Lists
| File | Description |
|------|-------------|
| `LinkedList_InsertAfter.java` | Insert a node after a given node (doubly linked) |
| `LinkedList_Remove.java` | Remove a node from a doubly linked list |

### 🌳 Trees
| File | Description |
|------|-------------|
| `Iterative_PreOrder_TreeTraversal.java` | Stack-based iterative pre-order traversal |
| `TreeHeight_PostOrder.java` | Recursive post-order tree height calculation |

### 📦 Data Structures
| File | Description |
|------|-------------|
| `Stack_ListBased.java` | Stack backed by a linked list |
| `Queue_ArrayV1.java` | Basic array queue with a fixed back pointer |
| `Queue_ArrayV2.java` | Circular array queue (space-efficient) |
| `MinHeap_Insert_SiftDown_Extract.java` | Min-heap with insert, sift-down, and extract-min |

### 🌐 Graphs
| File | Description |
|------|-------------|
| `IterativeDFS.java` | Depth-first search using an explicit stack |
| `Dijkstras_Algorithm.java` | Dijkstra's shortest path with a custom min-heap |

### ➗ Maths
| File | Description |
|------|-------------|
| `Euclids_GCD.java` | Euclid's algorithm for greatest common divisor |

---

## Notes

- All implementations are written from scratch — no built-in collections used for the core logic (except where a `Stack` or `ArrayList` is part of the scaffolding)
- Each file has a `main` method with a small test case so you can run it directly
- Code is kept intentionally straightforward — the goal is clarity over cleverness
